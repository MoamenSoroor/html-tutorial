Some Form Elements Attributes

created by Moamen Soroor at 26/6/2020 12:00 AM

Form attributes

Method: Get, Post
Action: the backend server file that process form elements
enctype: encoding of form default is application/x-www-form-urlencoded
novalidate: send data without validating any input in the form: like email or phone or specific pattern input.
autocomplete: can be set to on/off stop autocoplete feature in that form to prevent previous data from appear in new user inputs
target: like a tag links submit can be passed to another page or stay in the same page.

Disabled vs readonly

NOTE: that disabled form element doesn't sent to server. and doesn't appear in focus. In the other hand, readonly elements value is sent to server and is focusable


Method Get and Post

method = Get is the default and allow browser url to show form input vlaues, in the other hand, method = Post hide form elements from browser url

method = Get is the default and allow browser url to show form input vlaues, in the other hand, method = Post hide form elements from browser url

autocomplete on/off and autofocus on/off

autocomplete default is on, set it off to disable that property

autofocus default is on, set it off to disable that property

required attribute

form will not be submited until you assign a value to required fields

multiple attribute

By using multiple form elements attribute, you can add many items to the the element like: many tel , many email many files, by write your input and make a comma and space then enter the next value, note that many form elements doesn't affect with multiple attribute like date, time, button, text, and etc

novalidate attribute

By usin novalidate in form or in Submit or in image We can stop validation in form elements where there are a validation like: email, url, tel, or input with pattern.

Tie Element out of the Form

By usin form attribute of outer form element we can send that element to backend when we assign that form.

image input and submit input Form attributes

NOTE: we can assign form attribute to a submit or image input to allow sending of form to another backend with other submit options
form attributes that can be used at submit or image inputs are:

  • formaction
  • formmethod
  • formenctype
  • formtarget
